# SSAFY Start Camp 챗봇 퀘스트

광주_안현상, https://github.com/returnsweetpork

## I. 스펙(Specification)

------

구현된 어플리케이션의 주요 기능

#### (1) 번역 기능

- 사용자에게 '번역' 이후 내용을 자동 번역

#### (1) 얼굴인식 기능

- 사용자가 올린 사진을 자동으로 얼굴인식

#### (1) 로또 기능

- 사용자에게 랜덤한 6개 숫자를 넘겨줌

#### (1) 실시간검색어 기능

- 사용자에게 현재 실시간 검색어를 알려줌

## II. 회고(Retrospective)

------

어플리케이션 구현 과정에서의 어려움과 문제점

- 데이터를 크롤링 할 때 웹페이지 간의 차이 때문에 어려움이 있었다.

### III. 보완 계획(Feedback)

------

현재 미완성이지만 추가로 구현할 기능 및 기존 문제점 보완 계획

#### (1) 맛집추천 기능

- 가고 싶은 식당이 많을때 자동으로 랜덤한 맛집을 추천하여 결정해줌
