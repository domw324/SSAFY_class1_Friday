# SSAFY Start Camp 챗봇 퀘스트

광주_1_서상준, https://github.com/domw324

## I. 스펙(Specification)

------

구현된 어플리케이션의 주요 기능

#### (1) ㅇㅇㅇ 기능

- 사용자가 채팅을 통해 챗봇에게 전송한 데이터를 분석한다.
- 인물 사진을 입력한 경우, NAVER developer api를 이용하여 얼굴인식하여 반환
- "번역 <내용>"을 입력하면 papago를 이용해 번역하여 반환
- 데이터에 "점심"키워드가 들어간 경우, 랜덤으로 메뉴를 추출하여 반환
- "로또" 키워드가 들어간 경우, "자동"키워드 / "당첨" 키워드에 따라
  자동 번호를 내보내거나, 최신회차 추첨번호를 반환한다
- "실시간 검색어"를 입력하면 다음에서 크롤링하여 실시간 검색어를 반환

## II. 회고(Retrospective)

------

어플리케이션 구현 과정에서의 어려움과 문제점

- 데이터를 크롤링 할 때 크롤링을 막아 놓은 페이지가 있는 것에서 어려움이 있었다.

### III. 보완 계획(Feedback)

------

현재 미완성이지만 추가로 구현할 기능 및 기존 문제점 보완 계획

#### (1) 실시간 검색 사이트 선택 기능

- 실시간 검색어를 알고 싶을 때, 사이트를 선택하는 부분의 기능을 추가적으로 개발하고 싶다.
